# Sushi Bliss Reference Assets

The top-level `assets` directory exists to match the rebuild prompt structure.
The canonical reference screenshots are kept in `public/assets/screenshots` so
they can be loaded by the app and attached by Playwright without duplicate
files.

- `assets/screenshots`: general screenshot reference notes
- `assets/mobile-screens`: mobile reference notes
- `assets/tablet-screens`: tablet reference notes
- `assets/desktop-screens`: desktop reference notes
